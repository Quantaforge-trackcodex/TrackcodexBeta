# Character Image Placeholders

## Required Images
Create 12 character portrait PNG files in this directory (`public/assets/characters/`):

### Character List (lowercase filenames)
1. `blaze.png` - Orange warrior with fire theme (#FF6B35)
2. `aqua.png` - Cyan mage with water theme (#00D4FF)
3. `shadow.png` - Dark rogue (#2C3E50)
4. `steel.png` - Grey robot tank (#5DADE2)
5. `mystic.png` - Blue wizard (#3498DB)
6. `rose.png` - Pink support character (#FF69B4)
7. `frost.png` - Light blue ice warrior (#89CFF0)
8. `dusk.png` - Yellow mystic (#F39C12)
9. `ember.png` - Fire mage (#E74C3C)
10. `void.png` - Purple enigmatic character (#9B59B6)
11. `spark.png` - Orange/yellow support (#FF8C00)
12. `sage.png` - Green wise character (#27AE60)

### Specifications
- **Style**: Chibi-style (large head, small body)
- **Format**: PNG with transparency
- **Size**: 512x512 pixels recommended
- **Background**: Transparent
- **Theme**: Gaming/RPG aesthetic
- **Colors**: Match the hex colors listed above

### Also Required
- `default.png` - Fallback character image for unselected/error states

## Temporary Solution
Until custom character art is created, you can use:
1. Simple colored circles with character initials
2. Free game character sprites from sites like itch.io
3. AI-generated chibi character portraits
4. Geometric avatars with the theme colors

The system will fallback to `default.png` if a specific character image is missing.
